## 微信公众号爬虫

### 本程序监控微信流量

#### 打开公众号主页面 将这条接口推送到本程序
```
http://127.0.0.1:8887?url=
```

>

> PC端微信打开要爬取的公众号Home页面, 再打开本页面,然后网页看到公众号信息就可以了
